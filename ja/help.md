Each example consists of a complete small LaTeX document shown within
the page like this:

## English

English example:

```latex
\documentclass{article}
\begin{document}
Example text.
\end{document}
```

## Japanese

Japanese examples (only work in Overleaf currently) look like:

```latex
% !TEX platex
\documentclass{jsarticle}
\begin{document}
テキストの例。
\end{document}
```
